# PIR RDF plugin project API

## Env

Copy the ./API/src/main/resources/application.properties.example to the ./API/src/main/resources/application.properties and change variables (SMTP).

## Run locally

```sh
mvn spring-boot:run
```

If you would download the plugin, copy the build in ./plugin.

## Swagger

[Swagger locally](./src/main/resources/static/swagger.yml) or [Swagger in web](http://loaclhost:3001/swagger.html)
